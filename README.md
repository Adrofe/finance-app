# finance-app

This repository is a personal project to build a personal finance application.

## Database Infrastructure

See detailed database setup and environment variable documentation in [infra/db/README.md](infra/db/README.md).

## Installation

The following applications are required to run on Windows:

- Postgres
- Docker Desktop
- Git for Windows
- WLS

For installing WLS in Windows execute the following command in a powershell with admin privileges:

``` powershell
wsl --install
```
